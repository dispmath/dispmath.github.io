For Authors

Manuscript Requirements

Submissions should be written in English and prepared as a scholarly mathematics paper. Each manuscript should include a title, complete author information, an abstract, keywords, Mathematics Subject Classification 2020 codes, and a bibliography.

Authors should submit both the final PDF and the TeX source file through the submission form.

LaTeX Template

Authors are encouraged to prepare manuscripts using the Disputationes Mathematicae LaTeX style file and example template.

Download LaTeX template package

Submissions

To submit a manuscript, authors must send both the source .tex file and the compiled .pdf version to the Editor-in-Chief.

Submission of a manuscript is equivalent to:

  • confirming that the submitting author holds full copyright rights for this work and that the manuscript is original creative work.
  • declaring that there is no conflict of interest that could influence the submission or publication process.
  • confirming that the manuscript is not currently under consideration by another publication.
